Open in StackEdit is a simple Chrome extension I created to make it easy to open GitHub or GitLab directories in StackEdit, a popular online Markdown editor. I created it to make my life a bit easier, but it ended up gaining a few hundred other users as well, so that’s nice. It’s available on the Chrome Web Store here.
